# Instructions for Use

Clone the repo:
```
git clone https://github.com/midnightdoggo19/discord-bsky
```
Enter the new directory:
```
cd discord-bsky
```
Install dependencies:
```
npm i
```
Fill your Discord application's information in `example.env` and rename it to `.env`:
```
nano example.env
mv example.env .env
```
Run the bot:
```
node index.js
```
